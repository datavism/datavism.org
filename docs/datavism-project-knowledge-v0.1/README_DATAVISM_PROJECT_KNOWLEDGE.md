## Current Canon Summary

DATAVISM is an Artistic Research / Evidence Engine for AI-era data activism.

Core system:

- DATAVISM = The Evidence Engine
- G1 = Interactive Initiation
- Output = Signal Card
- Archive = collective living artwork
- Lines = modular, not forced
- Map recommends; user decides

## Product Priority

Do not prioritize completing the full 25-station curriculum first.

Prioritize the first vertical slice:

> A user enters G1 THE FOLDER, turns a vague suspicion into an investigable question, creates a Signal Card, and can optionally contribute an anonymized version to the Signal Archive.
